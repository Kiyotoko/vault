> (a + b) / 2 >= sqrt(a x b)
> (a - b)² >= 0
> a² + b² >= 2ab
> (a² + b²) / 2 >= ab

`min(a, b) <= 2 / (1/a + 1/b) <= sqrt(a x b) <= (a + b) / 2 <= sqrt((a² + b²)) / 2 <= max(a, b)`
