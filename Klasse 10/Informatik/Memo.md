# Datenbanken

## DR-Modell

| Form                     | Stellt dar      |
| ------------------------ | --------------- |
| Rechteck                 | Entitätentypen  |
| Eclipse                  | Attribut        |
| Raute                    | Beziehungen     |
| Unterstrichenes Attribut | Primärschlüssel |

## $N:M$ Beziehungen

Für jede $n:m$ Beziehung wird eine zusätzliche Tabelle angelegt, die die Fremdschlüssel beteiligten Tabellen enthält.
